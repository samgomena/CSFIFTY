This assignment includes several files:

- readme.txt 		 this file

- instructions.txt	 detailed assignment instructions

- background.txt 	 a (very) short story giving background and context
  			 for the assignment. This file is optional and you
  			 don't have to read it. It is just for fun.

- design_questions.txt	 the design questions that must be answered for the
  			 "design" portion of this assignment. 

- rovercontrol.cpp	 the main driver for this program. You must use this 
  			 for your main().

- scandata.h		 the definition for the scandata class.

- scandata.cpp		 the implementation for the scandata class.

- testqueue.h		 the test program for your queue data structure (optional)

- teststack.h		 the test program for your stack data structure (optional)

- commands.txt 	 	 the commands data file that you should use to do the
  			 final test of your program.

- expected.txt		 the expected results when you use commands.txt

- makefile		 a makefile that contains build and test targets (optional)
